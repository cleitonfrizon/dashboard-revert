## Tasks

2. [ ] **Link do projeto**: `vercel link --token=$VERCEL_TOKEN`

    ```bash
    vercel env add VITE_DASHBOARD_API_URL production
    vercel env add VITE_DASHBOARD_API_TOKEN production
    vercel env add VITE_DASHBOARD_PASSWORD_HASH production
    ```

4. [ ] **Adicionar domínio**: `vercel domains add dashboard.escalanegociosdigitais.com.br`
